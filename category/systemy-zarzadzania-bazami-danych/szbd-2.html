Ćwiczenia 2 Systemy zarządzania bazami danych

  1. Utwórz nową pustą bazę o nazwie PROG.
    • Kliknij prawym przyciskiem myszy na Database
    • Wybierz z listy New Database…
    • W polu Database name wpisz PROG
  2. Utwórz tabele w bazie PROG przy pomocy skryptu Lab2.1.PROG.struktura_bazy.sql (do pobrania na TEAMS)
    • Kliknij lewym przyciskiem myszy, w lewym górnym rogu na File
    • Z listy wybierz Open, a następnie File…
    • Wskaż lokalizację pobranego skryptu Lab2.1.PROG.struktura_bazy.sql i kliknij Open
    • Aby uruchomić skrypt, wciśnij klawisz F5 lub przycisk Execute
  3. Wygeneruj diagram bazy danych
    • Kliknij znak + obok utworzonej bazy danych PROG
    • Kliknij prawym przyciskiem myszy na Database Diagrams i wybierz z listy New Database Diagram
    • Trzymając klawisz shift, zaznacz lewym przyciskiem myszy wszystkie tabele i kliknij Add
    • Zamknij okno klawiszem Close
    • Zapisz diagram skrótem klawiszowym Ctrl + S
    • Utworzony diagram został zapisany w folderze Database Diagrams
  4. Wypełnij tabele danymi przy pomocy skryptu Lab2.2.PROG.dane.sql (do pobrania na TEAMS)
    • Postępuj tak samo jak w poleceniu nr. 2, wybierz plik Lab2.2.PROG.dane.sql
  5. Obejrzeć zawartość wszystkich tabel
    • Kliknij prawym przyciskiem myszy na nazwę tabeli
    • Aby podejrzeć zawartość tabeli, wybierz Select top 1000 Rows
    • Możesz modyfikować i dodawać rekordy za pomocą graficznego edytora, wybierając Edit Top 200 Rows
    • Możesz modyfikować właściwości kolumn, wybierając Design
  6. Napisz polecenia DML wykonujące następujące akcje. Jedno polecenie DML do każdego podpunktu (kliknij przycisk New Query, a następnie Ctrl+S aby zapisać plik z poleceniem):
    1. Dodanie siebie jako pracownika
    2. Dodanie trzech pracowników (dane dowolne, w jednym poleceniu!)
    3. Dodanie nowego stanowiska
    4. Aktualizacja danych pracowników dodanych w pkt 2. do stanowiska z pkt 3, i ustawienie pracownika z pkt 1. jako ich szefa. (jedno polecenie update!)
    5. Usunięcie wszystkich kar powyżej 500 zł
    6. Dodanie każdemu pracownikowi, poza prezesem, nagrody w kwocie 10% jego pensji
      1. https://www.tutorialspoint.com/sql/sql-sub-queries.htm
      2. TIP: INSERT INTO SELECT FROM WHERE
    7. Ustawienie wszystkim pracownikom pensji wyliczonej jako: dodatek * 2 + 1000
      1. TIP: UPDATE SET SELECT FROM WHERE
    8. Sprawdzenie czy jest w bazie stanowisko o nazwie „Programista”. Jeżeli jest zmienić mu dodatek na 1000. Jeżeli nie ma takiego stanowiska, dodać je, ustawiając kwotę dodatku na 1000.
      1. https://tutorialspoint.dev/language/sql/sql-merge-statement
    9. Napisz zapytanie (DQL) zwracające zestawienie w postaci: Imię, Nazwisko, Nazwa stanowiska, Suma należności. Gdzie suma należności jest wyliczona jako pensja + dodatek + nagrody + kary (zadanie dodatkowe!)
    10. Ustawienie prezesowi pensji na wartość o 100 większą niż najwyższa suma należności (pensja + dodatek + nagrody + kary) dla pracownika. (zadanie dodatkowe!)
      1. https://www.w3schools.com/sql/sql_min_max.asp
      2. https://www.w3schools.com/sql/sql_count_avg_sum.asp
