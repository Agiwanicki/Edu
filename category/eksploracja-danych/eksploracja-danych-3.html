K-means

Celem ćwiczenia jest graficzne przedstawienie efektu działania algorytmu K-means na przykładowym zbiorze współrzędnych.

  1. Utwórz nowy Repl Pythona o nazwie Clustering i zainstaluj biblioteki: 
    !pip install scikit-learn
!pip install pandas
!pip install numpy
!pip install matplotlib
  2. Na początku kodu zaimportuj biblioteki: 
    import sklearn
import pandas as pd
import matplotlib.pyplot as plt
from sklearn.cluster import KMeans
  3. Dodaj przykładowy zbiór współrzędnych x i y: 
    data = {
'x': [
25, 34, 22, 27, 33, 33, 31, 22, 35, 34, 67, 54, 57, 43, 50, 57, 59, 52, 65,
47, 49, 48, 35, 33, 44, 45, 38, 43, 51, 46
],
'y': [
79, 51, 53, 78, 59, 74, 73, 57, 69, 75, 51, 32, 40, 47, 53, 36, 35, 58, 59,
50, 25, 20, 14, 12, 20, 5, 29, 27, 8, 7
]
}
  4. Utwórz DataFrame z powyższego zbioru danych. Wyświetl współrzędne na wykresie za pomocą matplotlib. Uruchom program.
  5. Edytując wartość przypisaną do n_clusters, możesz określić, na ile grup algorytm ma podzielić dane. Zmienna centroids przechowuje listę punktów oznaczających „środki” wydzielonych grup.
  6. Wynik działania algorytmu zwizualizuj za pomocą biblioteki matplotlib. Centroidy zaznacz na czerwono. Każda grupa powinna stanowić odrębny kolor na wykresie.
  7. Podziel zbiór danych na co najmniej 4 grupy i obejrzyj wynik na wykresie.

K-neighbours

Celem ćwiczenia jest przyporządkowanie aut do odpowiednich kategorii za pomocą algorytmu K-neighbours.

  1. Wykomentuj kod dotyczący algorytmu K-means (poza importami). Dodaj nowe importy bibliotek na początku kodu: 
    from sklearn.neighbors import KNeighborsClassifier
from sklearn import preprocessing
  2. Pobierz z Teams pliki car.data oraz car.names. Umieść pliki w Colab i przeczytaj zawartość car.data instrukcją pd.read_csv("nazwa_pliku").
  3. Obejrzyj dane w pliku car.data. Nie wszystkie wartości zostały przedstawione numerycznie, np. kolumna buying zawiera wartości: v-high, high, med, low. Aby umożliwić dalszą analizę, przekonwertuj te wartości na numeryczne przy pomocy modułu preprocessing z biblioteki sklearn. Wyświetl dane po konwersji za pomocą instrukcji print(buying), a następnie wykomentuj tę instrukcję.
  4. Przypisz zmienne objaśniające do x i objaśniane do y. Do sformatowania wartości przypisanych do y wykorzystaj funkcję zip().
  5. Podziel dane na zbiór testowy oraz treningowy. Proporcje podziału określa wartość przypisana do test_size.
  6. Wykorzystanie KNN w kodzie Pythona jest bardzo zbliżone do regresji liniowej. Jedyna różnica polega na tym, że możemy określić, ilu „sąsiadów” ma szukać algorytm, poprzez przypisanie wartości do n_neighbors. Uruchom kod, aby zobaczyć model.score. Zmodyfikuj wartości test_size oraz n_neighbors, aby uzyskać jak najlepsze accuracy.
  7. Przypisz wynik predykcji do zmiennej predicted. Utwórz listę przechowującą klasy aut. Lista posłuży do „przetłumaczenia” nadanych wcześniej wartości numerycznych na opisy sprzed modyfikacji w punkcie 3.
  8. Za pomocą pętli for wypisz wynik prognozy, rzeczywistą klasę auta oraz dane, na podstawie których sporządzono prognozę klasy.
