Ćwiczenia 1 – Bazy Danych [0600-BFIF4B]

Wykonaj poniższe zadania z użyciem zapytań SQL. Pobierz skrypt instwind.sql z platformy Teams.

Za pomocą zapytań SQL

  1. Z tabeli Customers pozyskaj listę klientów z Polski.
    • SQL SELECT
    • SQL WHERE
  2. Z tabeli Customers pozyskaj listę imion klientów mieszkających w Berlinie.
    • Aby pozyskać same imiona, wpisz nazwę kolumny po SELECT.
  3. Znajdź nazwę najdroższego produktu w tabeli Products.
    • SQL ORDER BY
    • Aby ograniczyć liczbę rekordów do 1, dodaj LIMIT 1 na końcu zapytania.
  4. Znajdź numer ID najstarszego zamówienia w tabeli Orders.
  5. W tabeli Products zaktualizuj cenę produktu Chocolade do 30.
    • SQL UPDATE
  6. W tabeli Products podnieś cenę wszystkich produktów pochodzących od dostawcy Exotic Liquid o 10. Najpierw znajdź ID dostawcy w tabeli Suppliers.
  7. Utwórz tabelę ProductsInfo z kolumnami: ProductID (int), Note (varchar(200)), Warning (text).
    • SQL CREATE TABLE
  8. Oblicz średnią cenę produktu w tabeli Products.
    • COUNT, AVG, SUM
  9. Policz Niemców w tabeli Customers.
  10. Dodaj kolumnę isShipped (tinyint) do tabeli Orders.
    • SQL ALTER TABLE
