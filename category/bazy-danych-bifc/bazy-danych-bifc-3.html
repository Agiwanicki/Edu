Ćwiczenia 3 – Bazy Danych [0600-BFIF4B]

Wykonaj poniższe zadania za pomocą skryptów SQL.

  1. Załóż nową bazę o nazwie PROG2.
    CREATE DATABASE [PROG2]
  2. Załóż w bazie PROG2 strukturę tabel zgodnie z bazą PROG. Skrypt tworzący jej strukturę to Lab2.1.PROG.struktura_bazy.sql.
  3. Przepisz dane z bazy PROG do bazy PROG2 przy pomocy poleceń INSERT. Uwaga: należy użyć wyrażenia SET IDENTITY_INSERT.
    • SET IDENTITY_INSERT – przykład
    • Opis IDENTITY_INSERT
    SET IDENTITY_INSERT <nazwaTabeli> ON
INSERT INTO <nazwaTabeli>(kolumna1, kolumna2, ...)
SELECT kolumna1, kolumna2, ...
FROM nazwaBazy.nazwaTabeli
SET IDENTITY_INSERT <nazwaTabeli> OFF
  4. W bazie PROG2 utwórz tabelę INFO. Typy i rozmiary pól dobierz samodzielnie.
    • CREATE TABLE / IDENTITY
    • PRIMARY KEY
  5. Utwórz perspektywę umożliwiającą wprowadzenie informacji dla pracownika o karze.
  6. Analogicznie wykonaj wprowadzenie informacji dla pracownika o karze oraz dla szefa o karze.
