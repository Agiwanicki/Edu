Ćwiczenia 7 – Bazy Danych [0600-BFIF4B]

Temat: T-SQL – zmienne, instrukcje sterujące, pętle oraz funkcje.

Pobierz z Teams skrypt instwnd.sql. Uruchom skrypt w SSMS, aby odtworzyć bazę Northwind.

  1. Zmienne i operacje arytmetyczne
    • Typy danych T-SQL
    DECLARE @napis varchar(50) = 'Hello World!'
DECLARE @liczba int = 5, @liczba1 float = 0.5
SELECT @napis, @liczba

DECLARE @WYNIK int;
SET @WYNIK = @liczba + @liczba1
SELECT @WYNIK AS WYNIK

    Zwróć uwagę na konwersję typów (int + float → wynik typu int).

  2. Zmienne + zapytania na bazie Northwind 
    DECLARE @CustomerID nchar(5);
SELECT @CustomerID = CustomerID 
FROM Customers 
WHERE Country = 'Poland';

SELECT * FROM Orders WHERE CustomerID = @CustomerID;

DECLARE @SupplierID int;
SELECT @SupplierID = SupplierID 
FROM Suppliers 
WHERE Country = 'Germany';

SELECT * FROM Products WHERE SupplierID = @SupplierID;

    Uwaga: zapytanie przypisze tylko jedną wartość (ostatnią z wyniku).

  4. Pętla WHILE 
    DECLARE @counter AS INT = 1, @lname AS NVARCHAR(20);

WHILE @counter <= 5
BEGIN
    SELECT @lname = LastName 
    FROM Employees 
    WHERE EmployeeID = @counter;

    PRINT @lname;
    SET @counter += 1;
END;

    Porównanie z zapytaniem zbiorczym:

    SELECT LastName FROM Employees WHERE EmployeeID <= 5

    Bardziej złożony przykład:

    DECLARE @counter INT = 1, @price float = 0, 
        @stock INT = 0, @SUM float = 0;

WHILE @counter <= IDENT_CURRENT('Products')
BEGIN
    IF (SELECT UnitsInStock 
        FROM Products 
        WHERE CategoryID = 8 AND ProductID = @counter) IS NOT NULL
    BEGIN
        SELECT @stock = UnitsInStock 
        FROM Products 
        WHERE CategoryID = 8 AND ProductID = @counter

        SELECT @price = UnitPrice 
        FROM Products 
        WHERE CategoryID = 8 AND ProductID = @counter

        SET @SUM += @stock * @price
    END
    SET @counter += 1;
END;

SELECT @SUM

    To samo zapytanie w wersji optymalnej:

    SELECT SUM(UnitPrice * UnitsInStock) 
FROM Products 
WHERE CategoryID = 8
  5. Funkcja z zabezpieczeniem przed dzieleniem przez 0 
    CREATE FUNCTION dbo.Divide2(@x INT, @y INT)
RETURNS INT AS
BEGIN
    IF (@y != 0)
    BEGIN
        DECLARE @RESULT INT
        SELECT @RESULT = @x / @y
        RETURN @RESULT
    END
    RETURN 0
END
    SELECT dbo.Divide2(10, 0) AS Result
  6. Funkcja sprawdzająca parzystość liczby 
    CREATE FUNCTION dbo.CzyParzysta(@x INT)
RETURNS VARCHAR(30) AS
BEGIN
    IF(@x % 2 = 0)
        RETURN 'PARZYSTA'
    RETURN 'NIEPARZYSTA'
END
    SELECT dbo.CzyParzysta(2)
