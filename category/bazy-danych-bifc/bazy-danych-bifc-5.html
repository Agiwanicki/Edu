Ćwiczenia 5 – Bazy Danych [0600-BFIF4B]

Wykonaj poniższe zadanie za pomocą skryptów SQL w środowisku SQL Server oraz SSMS.

Implementacja projektu własnej bazy danych

Zaimplementuj projekt swojej bazy danych z wykorzystaniem SQL Server oraz SSMS. Baza danych musi zawierać:

  1. Przynajmniej 8 tabel dla zespołów 2-osobowych lub 5 tabel solo.
  2. Poprawne relacje między tabelami (klucze główne i obce).
  3. Więzy integralności oraz ograniczenia dziedziny (UNIQUE, ograniczenia liczby znaków, CHECK itp.).
  4. Przykładowe dane dodane za pomocą poleceń INSERT.
  5. Przykładowe zapytania DQL (SELECT).
