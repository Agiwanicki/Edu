Zajęcia – Funkcje skalarne, tablicowe, procedury i dummy data

Wykonaj poniższe przykłady w środowisku SQL Server Management Studio na bazie danych Northwind.

Celem ćwiczenia jest demonstracja działania funkcji skalarnych, funkcji tablicowych, procedur składowanych, w celu generowania przykładowych danych testowych (dummy data) w języku T-SQL.

Ograniczenia funkcji w T-SQL

  • W funkcjach nie można używać konstrukcji TRY ... CATCH.
  • W funkcjach nie należy używać RAND(), ponieważ jest to operator powodujący efekty uboczne.
  • Funkcja nie może rzucać wyjątków ani modyfikować danych poza własnym zakresem działania.

Przykład funkcji skalarnej

Funkcja skalarna zwraca jedną wartość. W poniższym przykładzie funkcja oblicza wiek na podstawie daty urodzenia. 

CREATE FUNCTION dbo.fn_ObliczWiek (@DataUrodzenia DATE)
RETURNS INT
AS
BEGIN
    DECLARE @Wiek INT;

    SET @Wiek = DATEDIFF(YEAR, @DataUrodzenia, GETDATE());

    IF (MONTH(@DataUrodzenia) > MONTH(GETDATE()))
       OR (MONTH(@DataUrodzenia) = MONTH(GETDATE()) AND DAY(@DataUrodzenia) > DAY(GETDATE()))
    BEGIN
        SET @Wiek = @Wiek - 1;
    END

    RETURN @Wiek;
END;

SELECT dbo.fn_ObliczWiek('1997-05-03') AS Wiek;

Przykłady funkcji tablicowych

Funkcja tablicowa zwraca tabelę, którą można wykorzystać np. w instrukcji SELECT, JOIN lub w procedurze składowanej. 

CREATE OR ALTER FUNCTION fn_ListaImion()
RETURNS TABLE
AS
RETURN
    SELECT *
    FROM (VALUES
        (1, 'Jan'), (2, 'Anna'), (3, 'Marek'), (4, 'Ewa'),
        (5, 'Piotr'), (6, 'Katarzyna'), (7, 'Tomasz'), (8, 'Agnieszka'),
        (9, 'Paweł'), (10, 'Magdalena'), (11, 'Grzegorz'), (12, 'Joanna'),
        (13, 'Michał'), (14, 'Zofia'), (15, 'Łukasz'), (16, 'Barbara'),
        (17, 'Adam'), (18, 'Maria'), (19, 'Rafał'), (20, 'Natalia')
    ) AS Imiona(ID, Imie);
CREATE OR ALTER FUNCTION fn_ListaNazwisk()
RETURNS TABLE
AS
RETURN
    SELECT *
    FROM (VALUES
        (1, 'Kowalski'), (2, 'Nowak'), (3, 'Wiśniewski'), (4, 'Wójcik'),
        (5, 'Kamiński'), (6, 'Lewandowski'), (7, 'Zieliński'), (8, 'Szymański'),
        (9, 'Woźniak'), (10, 'Dąbrowski'), (11, 'Lis'), (12, 'Król'),
        (13, 'Górski'), (14, 'Jankowski'), (15, 'Mazur'), (16, 'Kaczmarek'),
        (17, 'Krawczyk'), (18, 'Piotrowski'), (19, 'Grabowski'), (20, 'Michalski')
    ) AS Nazwiska(ID, Nazwisko);

Przykład procedury do generowania pracowników

Procedura składowana może wykonywać wiele operacji, w tym pętle, losowanie danych, pobieranie rekordów i wstawianie nowych wierszy do tabeli. 

CREATE PROCEDURE dbo.GenerujPracownikow
    @ile INT
AS
BEGIN
    DECLARE @i INT = 1;
    DECLARE @Imie NVARCHAR(50), @Nazwisko NVARCHAR(50);
    DECLARE @ImieID INT, @NazwiskoID INT;
    DECLARE @Tytul NVARCHAR(50) = 'Sales Representative';
    DECLARE @ReportsTo INT;

    WHILE @i <= @ile
    BEGIN
        BEGIN TRY
            SET @ImieID = CAST(RAND(CHECKSUM(NEWID())) * 20 + 1 AS INT);
            SET @NazwiskoID = CAST(RAND(CHECKSUM(NEWID())) * 20 + 1 AS INT);

            SELECT @Imie = Imie FROM dbo.fn_ListaImion() WHERE ID = @ImieID;
            SELECT @Nazwisko = Nazwisko FROM dbo.fn_ListaNazwisk() WHERE ID = @NazwiskoID;

            SELECT TOP 1 @ReportsTo = EmployeeID FROM Employees ORDER BY NEWID();

            INSERT INTO Employees (FirstName, LastName, Title, ReportsTo)
            VALUES (@Imie, @Nazwisko, @Tytul, @ReportsTo);

        END TRY
        BEGIN CATCH
            PRINT 'Błąd: ' + ERROR_MESSAGE();
            PRINT 'Błąd numer: ' + CAST(ERROR_NUMBER() AS NVARCHAR);
        END CATCH

        SET @i += 1;
    END
END;

EXEC dbo.GenerujPracownikow 5;

Poniżej znajdują się trzy podstawowe sposoby wstawiania danych do tabeli.

1. INSERT bez podawania nazw kolumn

Generowanie losowych wartości w T-SQL

W poniższym przykładzie tworzona jest prosta tabela testowa, a następnie losowana jest wartość ceny z określonego przedziału. 

CREATE TABLE ProduktyTestowe (
    ID INT IDENTITY(1,1) PRIMARY KEY,
    Nazwa NVARCHAR(100),
    Cena INT
);

DECLARE @Min INT = 10;
DECLARE @Max INT = 100;
DECLARE @LosowaCena INT;

SET @LosowaCena = FLOOR(RAND() * (@Max - @Min + 1)) + @Min;

INSERT INTO ProduktyTestowe (Nazwa, Cena)
VALUES ('Losowy produkt', @LosowaCena);
