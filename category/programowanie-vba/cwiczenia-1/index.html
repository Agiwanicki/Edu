Ćwiczenia 1 – Programowanie VBA

  1. Utwórz przycisk o nazwie Hello, który wyświetli komunikat Hello world! w MsgBox.
  2. Napisz procedurę, która doda do siebie dwie liczby podane przez użytkownika w InputBox.
  3. Napisz procedurę, która zapyta użytkownika o imię (InputBox), a następnie odpowie: „Witaj, imię!”.
  4. Utwórz przycisk, który po uruchomieniu wyświetli MsgBox z informacją, czy liczba z komórki C2 jest parzysta, nieparzysta, czy równa 0.
  5. Napisz procedurę, która wypisze w kolejnych komórkach kolumny A N kolejnych liczb. Wartość N podaj w InputBox.
  6. Wypisz w kolejnych komórkach kolumny A N pierwszych wyrazów ciągu Fibonacciego. Wartość N podaj w InputBox.

Dodatkowe uwagi

  • Zapisuj plik Excela jako .xlsm (arkusz z obsługą makr).
  • Aby włączyć kartę Deweloper: Plik → Opcje → Dostosowywanie wstążki → Deweloper.
  • Pierwszą procedurę dodasz w: Deweloper → Visual Basic → Insert → Module.
